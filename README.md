# BudgetMate

**BudgetMate** es una aplicación de consola en C++ para gestionar tus gastos personales.  
Permite registrar, visualizar y exportar tus gastos de manera sencilla y rápida.

## Funcionalidades

- Agregar gastos con **categoría, descripción y monto**  
- Ver un **resumen** del total de gastos  
- Mostrar todos los gastos en formato tabular  
- **Exportar** los datos a un archivo CSV (`gastos.csv`)

## Tecnologías

- Lenguaje: **C++**
- IDE recomendado: **VS Code**
- Compilador: **g++**

## Exportar a CSV
Al seleccionar la opción 4 en el menú, se genera un archivo gastos.csv en el mismo directorio del proyecto con todos tus registros.
